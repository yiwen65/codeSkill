# Example Project: Todo CLI

## Quick start

```bash
python3 example_project/src/cli.py add "学习 codeSkill"
python3 example_project/src/cli.py list
python3 example_project/src/cli.py done 1
python3 example_project/src/cli.py list
```

## Run tests

```bash
python3 -m unittest discover -s example_project/tests -p 'test_*.py'
```
